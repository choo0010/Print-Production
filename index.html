Production Knowledge

Preflight Standards
  • 300 dpi minimum for standard print
  • 150 dpi minimum for large format
  • 72 dpi for screen-only output
  • 3mm (1/8") standard bleed
  • CMYK for all print output
  • Embedded fonts or outlines
  • Verified image links
  • Overprint set on black text
  • PDF/X-1a or PDF/X-4 export
Colour Management
  • CMYK for offset & digital print
  • Pantone/PMS for spot colour
  • RGB → CMYK: review before press
  • 100K black for body text
  • Rich black: C60 M40 Y40 K100
  • Colour profile assignment on open
  • Soft proof before final export
  • Pantone bridge for cross-media
Finishing Awareness
  • Saddle-stitch: page count ÷ 4
  • Perfect bind: spine width calc
  • Fold zones: keep text clear
  • Cut tolerance: 3–5mm safety
  • Drill zone: no critical content
  • Perforation line layout rules
  • Large-format: generous bleed
  • Vinyl: surface prep awareness
File Formats
  • PDF / PDF/X — print standard
  • INDD — InDesign native
  • AI / EPS — vector print files
  • PSD — Photoshop composites
  • TIFF — high-res raster print
  • PNG / JPEG — digital output
  • SVG — scalable web/screen
  • Multi-format troubleshooting
